# Elevate Lab - Responsive Web Design

## Project Description
A simple responsive webpage built using HTML and CSS.  
The page includes a header, hero section, and footer with a clean layout.

## Technologies Used
- HTML5
- CSS3
- Flexbox
- Media Queries

## Features
- Responsive design for mobile, tablet, and desktop
- Structured layout using Flexbox
- Clean and modern UI

## How to Run
Open `index.html` in your browser or run using Live Server.
